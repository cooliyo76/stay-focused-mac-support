❓자주 묻는 질문 (FAQ)

Q. 포모도로 기법이란 무엇인가요?
25분 집중 작업 후 5분 휴식을 반복하는 시간 관리 기법입니다. 4회의 집중 세션 후에는 15분의 긴 휴식을 취합니다. 이 방법은 집중력을 유지하면서도 피로를 줄여 장시간 높은 생산성을 유지할 수 있게 해줍니다.
Q. 집중 시간과 휴식 시간을 변경할 수 있나요?
네! 설정 메뉴에서 집중 시간(1-60분), 짧은 휴식(1-30분), 긴 휴식(1-60분)을 자유롭게 조절할 수 있습니다. 개인의 집중력과 업무 스타일에 맞게 커스터마이징하세요.
Q. 통계 데이터는 어떻게 저장되나요?
모든 데이터는 사용자의 Mac에만 저장되며, 외부 서버로 전송되지 않습니다. 개인정보 보호를 최우선으로 하며, 완전히 오프라인으로 작동합니다.
Q. 세션 종료 시 알림을 받을 수 있나요?
네! 각 세션(집중/휴식)이 끝나면 macOS 알림을 통해 자동으로 알려드립니다. 첫 실행 시 알림 권한을 허용해주세요.
Q. 목표를 설정할 수 있나요?
하루 목표와 주간 목표를 각각 설정할 수 있습니다. 통계 화면에서 목표 달성률을 실시간으로 확인하고, 목표를 달성하면 축하 메시지를 받을 수 있습니다!
Q. 앱이 무료인가요?
네! Stay Focused는 완전 무료이며, 광고도 없고 구독도 필요 없습니다. 순수하게 생산성 향상만을 위한 앱입니다.
Q. macOS 버전 요구사항은?
macOS 14.0 (Sonoma) 이상에서 사용 가능합니다.
Q. 통계 기록을 초기화할 수 있나요?
통계 화면 하단의 "기록 초기화" 버튼으로 모든 기록을 삭제할 수 있습니다. 삭제 전 확인 메시지가 표시됩니다.

🔒개인정보 처리방침

최종 업데이트: 2026년 2월 20일

🛡️ 핵심 요약: Stay Focused Mac은 사용자의 개인정보를 수집, 저장, 전송하지 않습니다. 모든 데이터는 사용자의 Mac에만 저장되며, 완전히 오프라인으로 작동합니다.

1. 개인정보의 수집

Stay Focused Mac은 어떠한 개인정보도 수집하지 않습니다.

  • 이름, 이메일, 전화번호 등 개인 식별 정보를 수집하지 않습니다
  • 위치 정보를 수집하지 않습니다
  • 사용자 행동 추적을 하지 않습니다
  • 분석 도구나 추적 도구를 사용하지 않습니다

2. 데이터 저장

앱에서 생성되는 다음 데이터는 사용자의 Mac에만 로컬로 저장됩니다:

  • 포모도로 기록: 날짜별 집중 횟수 및 시간
  • 설정 정보: 타이머 시간 설정 (집중/휴식 시간)
  • 목표 설정: 일일/주간 목표 횟수

이 데이터는 macOS의 UserDefaults를 통해 사용자의 기기에만 저장되며, 외부 서버로 전송되거나 제3자와 공유되지 않습니다.

3. 데이터 전송

Stay Focused Mac은 인터넷 연결이 필요 없으며, 어떠한 데이터도 외부로 전송하지 않습니다:

  • 서버로의 데이터 전송 없음
  • 클라우드 동기화 없음
  • 제3자 서비스 통합 없음
  • 광고 네트워크 없음

4. 권한 요청

Stay Focused Mac은 다음 시스템 권한만 요청합니다:

  • 알림 권한: 포모도로 세션 종료 시 사용자에게 알림을 보내기 위함

이 권한은 사용자가 직접 허용/거부할 수 있으며, 거부하더라도 타이머 기능은 정상적으로 작동합니다. (단, 알림은 받을 수 없습니다)

5. 제3자 서비스

Stay Focused Mac은 어떠한 제3자 서비스도 통합하지 않습니다:

  • 분석 서비스 (Google Analytics 등) 미사용
  • 광고 네트워크 미사용
  • 충돌 리포팅 도구 미사용
  • 소셜 미디어 SDK 미사용

6. 데이터 보안

모든 데이터는 macOS의 보안 메커니즘에 의해 보호됩니다. 데이터는 사용자의 Mac 내부에만 저장되며, 앱 삭제 시 함께 제거됩니다.

7. 어린이 개인정보 보호

Stay Focused Mac은 모든 연령대가 안전하게 사용할 수 있습니다. 개인정보를 수집하지 않으므로 어린이의 개인정보 보호에도 문제가 없습니다.

8. 개인정보 처리방침 변경

개인정보 처리방침이 변경될 경우, 앱 업데이트 노트 및 이 페이지를 통해 공지합니다. 중요한 변경사항이 있을 경우 사전에 공지할 것입니다.

🛡️ 데이터 보호 원칙:
• 사용자 데이터는 Mac에만 저장 (UserDefaults 사용)
• 외부 서버로 데이터 전송 없음
• 사용자 추적 없음
• 광고 없음
